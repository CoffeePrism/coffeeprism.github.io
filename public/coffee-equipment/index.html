一、磨豆机——预算大头，先买它

为什么磨豆机这么重要

  • 研磨度均匀性 = 萃取均匀性 = 风味清晰度
  • 廉价电动磨豆机（< ¥300）：颗粒分布范围 200-1500 微米——同一杯咖啡里，细粉过萃 + 粗粉萃取不足同时发生
  • 好磨豆机（> ¥1500）：颗粒分布 600-900 微米——萃取干净均匀

手动磨豆机（手摇）

适合：单人喝、预算有限、喜欢仪式感、便携。

三、意式咖啡机（Espresso Machine）

关键决策：要不要买 espresso 机

先问自己：你会每天喝拿铁/卡布奇诺/平白咖啡吗？

  • 会 → 半自动机有意义
  • 不会，主要喝黑咖 → 不要买 espresso 机，手冲足够

意式机最大的隐藏成本不是价格，是学习曲线 + 配套器材 —— 一台好磨豆机 + 压粉器 + WDT 针 + 奶缸 + 蒸汽棒清洁工具，加起来又一笔。

八、常见决策困惑

「我应该买好磨豆机还是好咖啡机？」

好磨豆机。一台 ¥3000 磨豆机 + ¥3000 咖啡机 ≫ ¥1000 磨豆机 + ¥5000 咖啡机。

「全自动机值得吗？」

仅在你绝对不想学习+预算 ¥15,000+ 的前提下值得。否则半自动 + 一台好磨豆机更值。

「胶囊咖啡呢？」

作为「速食咖啡」的升级版可以，作为咖啡爱好的起点不行——你失去了所有控制变量。

「Hario 还是 Kalita？」

  • 喜欢清晰明亮 → Hario V60
  • 喜欢醇厚平衡、容错率高 → Kalita Wave
  • 想都试 → 都买，加起来才 ¥400
