# 第三章 研磨的艺术

在咖啡所有变量里，**研磨度是对味道影响最大的一个**。温度改变 2℃ 你可能喝不出，但研磨度多拧一格，这杯咖啡就从"酸甜平衡"变成"干涩泛苦"。理解研磨，你才真正握住了手冲的方向盘。

## 粒径分布（PSD）：不是"平均粗细"这么简单

实验室用激光粒度仪把咖啡粉测一遍，会得到一个**粒径分布（Particle Size Distribution, PSD）**曲线——横轴是微米（µm）、纵轴是颗粒数量或体积占比。

这条曲线的形状比"平均粒径"重要得多。两台磨豆机可以有**完全相同**的平均粒径（比如都是 750µm），但曲线形状差异巨大——一台是**单峰且窄**（所有颗粒都在 600–900µm），另一台是**双峰**（一半在 800µm 附近、另一半散布在 100µm 的细粉 fines 和 1400µm 的粗粒 boulders）。

**萃取结果会完全不同。**

### 单峰窄分布（Unimodal, Narrow）

- 每一颗咖啡粉的萃取速率接近
- 水穿过粉床阻力均匀，不易 channel
- **风味干净、层次分明、甜感好**
- 平刀磨豆机（尤其是高端如 EK43、Weber Key）更容易产生这种分布

### 双峰分布（Bimodal）

- 大部分颗粒在目标区间，但有显著的 fines（<100µm）
- fines 因表面积巨大，**萃取极快**且过度
- boulders（>1200µm）因内部扩散慢，**萃取严重不足**
- 结果：**同一杯里既有 over 又有 under**——味道复杂但也更"浑浊"

锥刀磨豆机大多是双峰，不意味着"不好"，只是**风格不同**。有人就喜欢锥刀那种"有 body"的质感。

## Fines 与 Boulders：罪魁祸首？

### Fines（细粉，<100µm）

- 萃取极快（~10 秒就溶出大量物质）
- 过多的 fines 会**堵塞滤纸孔隙**导致 drawdown 变慢、过萃
- 是**"muddy taste"**（浑浊苦涩）的主要元凶

### Boulders（粗粒，>1200µm）

- 萃取极慢
- 在手冲的短时间（2:30–3:30）内几乎只萃取到酸，没到甜
- 导致**酸尖、风味单薄**

### Fines 与 Boulders 的处理

完全消除不可能。可接受做法：
- **过筛（sifting）**：用 Kruve、Normcore 筛网，去掉 <200µm 和 >1000µm 的两端。比赛选手常用。
- **降低研磨速度**：手摇磨不要太快，电动磨大部分无关（但有些会因热量产生静电吸粉）
- **刀盘对齐（alignment）**：DF64 之类的电动磨，对齐后的 PSD 更集中

## 手冲的目标微米范围

这是 2026 年业界基本共识（不是"真理"，但这些数是有实验数据支撑的）：

| 场景 | 微米范围 | 说明 |
|---|---|---|
| **浅烘、V60、2:30–3:30** | 600–800 µm | 多数精品浅烘的舒适区 |
| **中烘、V60** | 700–900 µm | 更粗，避免大分子苦物 |
| **深烘、V60** | 850–1000 µm | 深烘易过萃，需粗一档 |
| **Kalita / 平底** | 较 V60 粗 50–100 µm | 平底接触时间长，研磨需粗 |
| **Chemex** | 900–1100 µm | 接近法压粗度 |
| **冠军比赛配方（浅烘）** | 约 650–750 µm | 多数配方偏细 |

**如何"知道"自己的微米数**？要么买一台**粒径分析仪**（不现实），要么**参考网上社区对你这台磨豆机的刻度 → 微米对照表**（reddit r/espresso、Home-Barista 都有）。

## 主流手冲磨豆机参考参数

### Timemore C2 / C3 Pro

- 38mm **S2C 锥刀**
- C3 Pro 手冲推荐刻度：**22–26 格**（大致对应 700–900µm）
- 入门性价比之王，预算 500 元内

### 1Zpresso JX-Pro / JX

- **48mm 锥刀**（比 C2/C3 大一圈，切割效率高）
- 400 格微调（每格 12.5µm 左右，根据型号）
- 手冲推荐：**外环 2 格 + 内环 60–100 格**（具体需参考你的型号说明）
- JX-S Pro 针对浅烘有调整
- **官方刻度表具体数值需参考设备说明书**

### 1Zpresso ZP6 / ZP7

- **平刀**手摇，专为手冲设计
- 粒径分布比锥刀更窄，更"EK43 风格"
- 手冲推荐：**约 60–90 格**（具体需参考配方）

### Comandante C40 MK4

- **47mm 锥刀**（Nitro Blade 氮化钢）
- 手冲推荐：**22–30 格**（从基础位 0 向松方向转动格数）
- 德国制造，二手保值，社区共识"退烧机"之一

### 电动推荐范围速览

- **DF64 / DF64P（改装后）**：手冲范围约 4.5–6.5（刻度），需参考刀盘
- **Timemore Sculptor 078**：手冲范围约 2.5–4.0 转
- **Weber Key**：数字刻度 60–75
- **EK43 / EK43s**：手冲范围约 9–11（取决于状态）

> 所有刻度都是"参考值"——**每一台磨豆机都有个体差异**，同一型号新机和用了 500kg 之后的 PSD 不完全一样。**自己校准**比背刻度重要（见第八章）。

## 电动 vs 手动：什么场景选哪个

### 手动磨豆机的真实场景

- **出差、露营、办公室**：无需插电，100% 便携
- **小份量（10–25g）**：手摇 90 秒以内，不累
- **预算敏感**：同样 2000 元，手动能买到 1Zpresso JX-Pro，电动只能买到入门
- **噪音敏感**：公寓早起不吵醒家人

### 电动磨豆机的真实场景

- **多杯（>30g/次 或一天多次）**：手摇会累
- **家中有多人喝咖啡**
- **追求更高 PSD 质量**：高端电动平刀（EK43、Weber、Mahlkönig）超过任何手摇
- **用意式 + 手冲兼顾**：直接用大磨一台多用

**典型升级路线**：Timemore C2 → 1Zpresso JX-Pro → DF64 → Weber Key / EK43。不必每步都走。

## 刀盘保养与更换

- **每次冲煮前**：用毛刷或吹气清理出粉口
- **每月一次**：完全拆解（电动磨需断电），刷净刀盘、磨仓、粉仓
- **每半年一次**：深度清洁（用 Grindz 磨豆机清洁颗粒，按产品说明使用）
- **刀盘更换周期**：
  - 锥刀通常 **500–1500kg** 咖啡豆寿命
  - 平刀通常 **500–1000kg**
  - EK43 级别刀盘 **1500–3000kg**

家庭用户一年不过 10–30kg 咖啡豆，刀盘寿命按**小时数**反而更合适——用到刀盘开始**钝化（cut → crush）**，研磨 PSD 明显变差、fines 增加，即需更换。

**钝刀的味觉信号**（见第七章）：
- 同刻度，味道突然变"平"
- 明显苦味上升，甜感下降
- drawdown 变慢

## 残粉（Retention）：被低估的变量

"残粉"指磨豆机内部卡住、本次没被磨出的咖啡粉。下次磨的时候，**这部分陈粉会被新粉推出来**，混进你的杯中。

### 典型残粉量

- **手动锥刀**：0.2–1.0g
- **单剂量电动平刀（DF64 单剂量改装）**：0.1–0.5g
- **传统电动定量机（带粉仓）**：2–10g（非单剂量设计）
- **EK43（无改装）**：1–3g

### 为什么重要

- **跨豆子冲煮**：今天浅烘、明天深烘，残粉会串味。第一杯咖啡可能"怪怪的"。
- **称重不准**：你以为磨了 20g 出来 19.5g，实际 20g 进、18.7g 出——因为 1.3g 进了残粉，又有 1.1g 陈粉挤出来——算进杯中的"新粉"只有 17.6g。

### 降残粉技巧

- **RDT（Ross Droplet Technique）**：磨豆前喷几滴水到豆子上，减少静电、fines 附着
- **单剂量清仓（bellow）**：每次磨完用气囊吹出残粉
- **跨豆清仓**：换豆子时先用 2–3g 新豆"清仓"，扔掉

---

**本章要点**

- 研磨看的是粒径分布（PSD），不是平均粗细；窄分布 = 干净、双峰 = 厚实
- Fines（<100µm）过萃是浑浊味的元凶；boulders（>1200µm）欠萃是酸尖瘦杯的元凶
- 手冲目标微米范围大致 600–900µm，具体视豆子和滤杯
- 刻度 → 微米每台磨都不同，自己校准比背数字重要
- 手动适合单人、便携、预算；电动适合高频、多人、极致 PSD
- 刀盘有寿命，钝刀味觉信号：味道"平"、苦升甜降、drawdown 变慢
- 残粉影响一致性，RDT + 单剂量清仓 + 跨豆清仓是常规处理
