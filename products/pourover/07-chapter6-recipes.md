# 第六章 冠军配方详解

这一章给出**过去十年最具影响力的手冲配方**的具体参数、时间轴、以及"这套配方**在优化什么**"的分析。所有参数来自各自选手的公开发布（YouTube、WBrC 官方、个人网站、Barista Hustle 采访等）。比赛现场会根据豆子微调，**参数视为参考基线，不是圣经**。

---

## 1. Tetsu Kasuya 4:6 Method（2016 World Brewers Cup 冠军）

### 基础信息

- **滤杯**：Hario V60 02（塑料或陶瓷）
- **滤纸**：Hario tabbed
- **粉重**：20g
- **水量**：300g（粉水比 1:15）
- **水温**：93℃（浅中烘）/ 88℃（深烘）
- **研磨**：**中粗**（约 800–900µm，比常规 V60 粗一档）
- **总时间**：约 3:30

### 时间轴

| 时间 | 动作 | 累计水量 |
|---|---|---|
| 0:00 | 第 1 段注水 60g（中心→外圈螺旋） | 60g |
| 0:45 | 第 2 段注水 60g | 120g |
| 1:30 | 第 3 段注水 60g | 180g |
| 2:10 | 第 4 段注水 60g | 240g |
| 2:30–2:50 | 第 5 段注水 60g | 300g |
| 3:30 | Drawdown 完成 | 300g |

### 这套配方在优化什么

- **可调性**：4/6 比例让你在不改变研磨的情况下调酸甜
- **容错**：5 段注水每段有缓冲时间，新手也能控制
- **适配性**：对浅、中、深烘都能通过水温和 4/6 比例适配
- **复现性**：粉水比、时间点都是整数，秤即可

## 2. Emi Fukahori 2018 WBrC 冠军配方

Emi Fukahori（瑞士）使用 **Kalita Wave 185**，围绕 **Bypass**（预冲 + 稀释）设计。

### 基础信息

- **滤杯**：Kalita Wave 185
- **滤纸**：Kalita Wave 185 原装
- **咖啡豆**：埃塞 Hambela 水洗（赛场用豆）
- **粉重**：20g
- **冲煮水量**：240g（粉水比 1:12，偏浓）
- **Bypass 水**：60–80g 热水，赛后加入咖啡液中
- **水温**：约 91℃
- **研磨**：中等偏细
- **总冲煮时间**：约 2:30

### 时间轴（简化）

| 时间 | 动作 |
|---|---|
| 0:00 | 闷蒸 60g（3× 粉重） |
| 0:30–0:45 | 等粉床 |
| 0:45 | 主注水到 240g（分 2–3 段） |
| 2:30 | Drawdown 完成 |
| 赛中 | 将约 60–80g 热水直接兑入萃取液 |

### 这套配方在优化什么

- **浓度 + 清爽并存**：1:12 高浓缩拿到风味前中段，稀释避免后段苦涩
- **TDS 控制**：bypass 把最终 TDS 稳定在 1.25% 左右
- **对浅烘高酸质豆**：bypass 结构让花果香突出、不被稀释带走的同时避免过萃

> **具体每段水量、时间点需参考 Emi Fukahori 公开演示视频和 WBrC 官方赛后资料。**

## 3. Jiwon Jung 2019 WBrC 冠军配方

Jiwon Jung（韩国，Coffee Graffiti）以 **Origami + 极低水温 + 预浸泡** 组合拿到冠军，开启了"冷水冲浅烘"的思路。

### 基础信息

- **滤杯**：Origami（配 Kalita Wave 185 滤纸）
- **粉重**：20g
- **水量**：300g（粉水比 1:15）
- **水温**：**约 70℃**（极低！）
- **研磨**：细（约 600µm 或更细）
- **总时间**：约 3:00–3:30

### 时间轴

| 时间 | 动作 | 累计 |
|---|---|---|
| 0:00 | 闷蒸 60g | 60g |
| 0:45 | 持续注水至 300g | 300g |
| 3:00–3:30 | Drawdown | - |

### 这套配方在优化什么

- **低温 + 细研磨 + 长时间**：用"细研磨 + 长时间"补偿低温的低 k（Arrhenius），专门萃取**酸甜前段**
- **Origami + Kalita 纸组合**：流速介于 V60 和 Kalita 之间，Jung 认为对浅烘最平衡
- **结果**：**极致花果香、柔和的酸、几乎无苦**——评审对"新感知"给高分

> 在家复现需要注意：**低水温要求豆子本身品质极高**（比赛豆级别），普通豆子 70℃ 会严重欠萃。普通玩家可尝试 80–83℃。

## 4. Matt Winton 2021 方法

Matt Winton（瑞士，2021 年 World Brewers Cup 冠军）代表"**极简 + 连续注水**"流派。

### 基础信息

- **滤杯**：Orea Brewer V3（或 Kalita Wave）
- **粉重**：14g
- **水量**：234g（粉水比 1:16.7）
- **水温**：93℃
- **研磨**：中细（根据 Orea 特性偏细，约 600–700µm）
- **总时间**：约 2:30–3:00

### 时间轴

| 时间 | 动作 | 累计 |
|---|---|---|
| 0:00 | 闷蒸 45g（约 3× 粉重） | 45g |
| 0:45 | 连续注水（约 6g/秒）至 234g | 234g |
| 约 1:17 | 注水完成 |  |
| 2:30–3:00 | Drawdown 完成 | - |

### 这套配方在优化什么

- **减少手法干预**，让豆子本身的特色出来
- **连续注水的流体动力学**：整段粉床持续悬浮、fines 不会聚集到底部
- **配合快流速器具（Orea）**：避免连续注水堵塞
- **稳定性极高**：没有多段切换，不怕秤抖、水温下降

## 5. Lance Hedrick 配方

Lance 没有单一"签名配方"，下面给出他 YouTube 频道反复演示的两套。

### 5a. Lance "Ultra" V60（难萃浅烘救命配方）

- **滤杯**：V60 02
- **滤纸**：Cafec Abaca Light Roast
- **粉重**：15g
- **水量**：250g（1:16.7）
- **水温**：96℃（几乎沸点）
- **研磨**：细（约 550–650µm，"espresso 粗 + 手冲细"之间）

**时间轴**：

| 时间 | 动作 | 累计 |
|---|---|---|
| 冲煮前 | 粉碗内 WDT 均粉 | - |
| 0:00 | 闷蒸 60g（4× 粉重） | 60g |
| 0:00–0:10 | 轻晃滤杯（swirl）2 次 | - |
| 0:45 | 注水至 150g | 150g |
| 1:15 | 注水至 250g | 250g |
| 1:30 | 最后 Rao Spin | - |
| 2:30 | Drawdown 完成 | - |

### 5b. Lance Orea 快冲配方

- **滤杯**：Orea V3 + Sibarist Fast 滤纸
- **粉重**：20g
- **水量**：300g
- **水温**：95℃
- **研磨**：非常细（≈ 500µm）
- **总时间**：**约 1:30–2:00**

**时间轴**：
- 闷蒸 50g，30 秒
- 一次性倾入剩余 250g（约 15–20 秒完成）
- Drawdown 至 1:45

**优化**：超快流速 + 超细研磨 + 高温，用"研磨度"而非"时间"决定萃取。

## 6. James Hoffmann V60 配方（YouTube 8M+ 播放量）

2019 年 James Hoffmann（2007 WBC 世界咖啡师冠军）发布的"Ultimate V60 Technique"视频是全球观看最多的手冲教学。

### 基础信息

- **滤杯**：V60 02（材质任选）
- **滤纸**：Hario tabbed
- **粉重**：30g
- **水量**：500g（粉水比 1:16.7）
- **水温**：**刚离火后自然降到约 93℃**
- **研磨**：中细（略比"table salt"粗）
- **总时间**：**3:30–4:00**

### 时间轴

| 时间 | 动作 | 累计 |
|---|---|---|
| 0:00 | 注水 60g 闷蒸（2× 粉重） | 60g |
| 0:00–0:10 | **Swirl 轻晃滤杯**（不搅拌） | - |
| 0:45 | 开始注水，到 1:15 注完至 300g | 300g |
| 1:15–1:45 | 缓慢注水至 500g | 500g |
| 1:45 | **再次 Swirl 轻晃一次** | - |
| 3:30–4:00 | Drawdown 完成 | - |

### 这套配方在优化什么

- **极简易记**：60g 闷蒸、到 300g、到 500g，两次 swirl
- **可重现性**：James 本人反复实测，不同烘焙度都能用
- **swirl 替代搅拌**：避免 WDT 带来的不稳定
- **适配家庭**：30g 粉量是家用最舒适的尺寸

### Hoffmann 的强调

1. **预热、预热、再预热**——滤杯、下壶都要
2. **水温宁高勿低**——宁可 94℃ 也不要 88℃
3. **研磨调到 drawdown 刚好在 3:30–4:00**——研磨度比水温更重要

> 这套配方被称为"家庭黄金配方"的原因：参数极易记住、容错高、在各种豆子上不出大错。

## 7. 配方选择决策树

什么时候用哪套？

| 场景 | 推荐 |
|---|---|
| **新手想稳定** | James Hoffmann V60 |
| **想按豆子微调酸甜** | Tetsu Kasuya 4:6 |
| **有 Kalita Wave、想尝试 bypass** | Emi Fukahori |
| **极浅烘、花果香追求者** | Jiwon Jung（降温法） |
| **有 Orea / April，想省事** | Matt Winton 连续注水 |
| **浅烘难萃、豆子"没发展"** | Lance Ultra（高温、细磨、大闷蒸） |

---

**本章要点**

- Kasuya 4:6 是最可调的通用配方，适合按豆子微调
- Emi Fukahori 用 bypass 结构在 Kalita Wave 上实现"浓且干净"
- Jiwon Jung 用 70℃ 极低温 + 细磨 + Origami 拿到极致花果香
- Matt Winton 用连续注水 + 快流速器具（Orea）做"减法手法"
- Lance Hedrick 的两套配方代表"大闷蒸 + 高温" 和 "超快流速 + 超细研磨"
- James Hoffmann V60 是家庭黄金配方，3:30–4:00 drawdown 是核心目标
- 所有配方都是"参考基线"，根据豆子和器具**需微调再复现**
