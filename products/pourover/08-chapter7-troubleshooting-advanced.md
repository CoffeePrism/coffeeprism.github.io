# 第七章 高级故障排查

到了高级阶段，"我的咖啡不好喝"这句抱怨已经不够用了。你需要**更细的诊断词汇**和**更系统的调参路径**。这一章给你一张**症状 → 原因 → 修正**的诊断地图，以及每一种高级问题的讲解。

## Channeling（通道效应）：侧边 vs 中心

**定义**：水在粉床里找到一条"捷径"通过，不均匀穿过所有粉。结果是**局部过萃 + 整体欠萃**——极糟糕的味道组合。

### 侧边 Channeling（Side Channel）

**症状**：
- 粉床侧壁（滤纸贴壁位置）出现**干区**或**凹陷**
- drawdown 末期滤纸上留下**粉的划痕**
- 味道**尖锐酸 + 淡**

**常见原因**：
- 注水靠近边缘，水直接从滤纸内壁"贴壁而下"
- 滤纸未 rinse 湿透，某侧粘粉更紧
- 粉量太少，粉床过浅

**修正**：
- 注水保持**中心 2–3cm 区域**，不要接近边缘 1cm 以内
- Rinse 用热水彻底浸湿滤纸
- 确保粉量够（V60 02 最少 12g，理想 15–20g）

### 中心 Channeling（Center Channel）

**症状**：
- 粉床中心出现**明显的注水凹坑**
- 坑里可能看到水层持续存在
- 味道**同样酸 + 淡**

**常见原因**：
- 注水过猛、水柱太粗、水速太快
- 粉床未均匀铺平
- Fines 在中心堆积

**修正**：
- 降低水柱流速（细口鹅颈壶）
- 注水前用手指敲杯边、或做一次 **WDT** 均粉
- 注水开始时先从**中心小圈**向外**缓慢螺旋**

## 读粉床：冲完之后能告诉你什么

**优质的冲后粉床**：
- 整体**平整如盘**或**略微凹陷**
- 侧壁不粘残粉、不留划痕
- **颜色均匀**，无局部深色（代表局部过萃）

**异常粉床**：
| 现象 | 解读 |
|---|---|
| 中心凹陷深坑 | 中心 channeling |
| 边缘粘纸、干块 | 侧边欠萃 |
| 整体倾斜（一侧高一侧低） | 注水不均匀 / 滤杯不水平 |
| 表面水珠 | drawdown 太慢（粉太细或 fines 过多）|
| 一块一块（crater，火山口） | 闷蒸期 CO₂ 爆发不均 |

## Drawdown 速度：过快与过慢

**理想 drawdown**：注水完成后，约 30–60 秒内完全滤出。总时间 2:30–3:30（因配方不同略有差异）。

### Drawdown 过快（< 30 秒）

**现象**：注水完立即全漏空。味道**尖酸、稀薄、无甜感**。

**原因 → 修正**：
| 原因 | 修正 |
|---|---|
| 研磨过粗 | 研磨细一格 |
| 水温过低 | 升温 2–3℃ |
| 粉量不足 | 增加粉（至 15g+） |
| 滤纸本身流速过快（如 Sibarist Fast） | 更换常规滤纸或调细研磨 |
| Channeling | 见上节 |

### Drawdown 过慢（> 60 秒）

**现象**：水滞留粉床，滤不下来。味道**苦、涩、浑浊、死甜**。

**原因 → 修正**：
| 原因 | 修正 |
|---|---|
| 研磨过细 | 研磨粗一格 |
| Fines 过多（Grinder PSD 差） | 过筛或升级磨豆机 |
| Fines migration（搅拌过度） | 减少搅拌 |
| 滤纸堵塞（低质量纸） | 换 Cafec / Hario 原装 |
| 滤杯未预热，水降温过多 | 充分预热 |

## Fines Migration：被低估的问题

**定义**：细粉在萃取过程中向下迁移，在**粉床底部形成致密层**，堵塞滤纸孔。

### 判断 Fines Migration

- 注水期 drawdown 正常，**最后 20%** 突然变慢
- 粉床底部（揭开滤纸可见）有**黏腻致密层**
- 味道：有 body 但**后段苦涩**

### 修正

- 减少搅拌次数（最多 1 次 Rao Spin）
- 降低水柱流速（stream 过猛会搅乱 fines）
- 磨豆机过筛（Kruve 筛去 <200µm）
- 升级磨豆机（平刀 > 锥刀，PSD 集中）

## 刀盘磨损的味觉信号

刀盘用到中后期（家庭用户大概 2–3 年、每日用）会出现**隐性钝化**。不是"磨不动了"，而是 PSD 变差：fines 增多、boulders 也增多（"把豆子砸碎"而非切开）。

### 信号

- **同刻度味道变平**：酸、甜、苦都变弱，杯子"灰色调"
- **苦明显上升**：fines 增多，过萃
- **drawdown 变慢**：fines 堵塞
- **之前好喝的配方突然不灵**：需要研磨粗一档才能复现

### 验证

- 换一次同价位/同级别新刀盘，对比 3 天
- 或用另一台磨磨同一支豆子对比

### 更换

- 锥刀：500–1500kg（家用 = 3–10 年）
- 平刀：500–1000kg
- 家用年消耗 10–30kg，**用味觉判断**比按年限判断靠谱

## 水温不一致问题

### 常见来源

1. **廉价水壶没有温控**，烧水后温度快速下降，实际注水时水温比你以为的低 5–8℃
2. **Fellow Stagg 等温控壶**保温功率有限，长时间（> 3 分钟）不使用，液面温度会比设定低 1–2℃
3. **冬季室温低**，水降温快
4. **滤杯未预热**，第一段注水时粉床温度比水低 3–5℃

### 修正

- 温控壶 + 预热所有器具（滤杯、下壶、分享杯）
- 如果没温控壶：**水壶备个温度计**，按读数调整
- 冬天可以设置比目标温度高 2–3℃
- 如果想极致：用**接触式温度计**测第一段注水后的粉床液面温度，建立自己的"壶嘴 → 粉床"温差模型

## 滤纸味（Paper Taste）

**症状**：咖啡入口有**纸浆、纸板、木质**的收敛干感。浅烘高酸质豆子上尤为明显。

### 原因

- 滤纸未 rinse
- 滤纸品质差（未漂白 + 薄）
- 滤纸受潮、吸了异味（油烟、调料）

### 修正

- 用**至少 150g 热水** rinse
- 换漂白滤纸（Hario tabbed / un-tabbed）
- 浅烘推荐 **Cafec Abaca**（马尼拉麻纤维，纸味几乎无）
- 滤纸装**密封罐**保存，远离油烟

## 海拔影响：高海拔 = 沸点低

海拔每升高 1000m，水的沸点降低约 3℃。2500m 海拔地区，水烧开是 91.5℃ 左右。

### 影响

- **同样"烧开"的水，高海拔萃取力更低**
- 需要的水温调整：
  - 低海拔（< 500m）常规参数
  - 中海拔（1000–2000m）水温加 2–3℃（实测设定）
  - 高海拔（> 2500m）"烧开"可能不够，需要更细研磨 + 更长接触时间补偿

### 修正

- 研磨细一档
- 闷蒸时间加长 10 秒
- 总冲煮时间加长 20–30 秒
- 某些高海拔咖啡馆用**压力水壶**提升沸点

## "Muddy"（浑浊）味的成因

"Muddy"在杯测术语里指**浑浊、模糊、层次不清**的负面特征。常见组合原因：

1. **Fines 过多 + 过萃**：研磨太细或刀盘钝化
2. **水质碱度过高**：缓冲度压住酸，苦涩浮现但酸不平衡
3. **水温过高 + 深烘**：过度萃取大分子
4. **Fines migration + 搅拌过度**
5. **陈豆**：烘焙后 > 60 天，氧化产物充斥杯中

诊断顺序：换水 → 清理磨豆机 → 降水温 → 减少搅拌 → 换豆子。

## 症状 → 原因 → 修正 快查表

| 症状 | 最可能原因 | 修正优先级 |
|---|---|---|
| **酸尖、薄、无甜** | 欠萃 | 研磨细 → 升水温 → 延时 |
| **苦、涩、挂舌** | 过萃 | 研磨粗 → 降水温 → 缩时 |
| **浑浊 muddy** | Fines / 水 / 陈豆 | 清磨、换水、换豆 |
| **Drawdown < 2:00** | 研磨粗/粉少/温低 | 细研磨 |
| **Drawdown > 4:00** | 研磨细 / fines / 堵 | 粗研磨 + 减搅拌 |
| **粉床中心凹坑** | 注水过猛 | 换细口壶、降流速 |
| **粉床侧壁粘粉** | 侧 channel | 中心注水、充分 rinse |
| **味道"灰"、无层次** | 刀盘钝 / 陈豆 / 水差 | 换刀盘 / 换豆 / 换水 |
| **第一口没问题，后味苦** | Fines migration | 减搅拌、过筛 |
| **纸板/收敛感** | 滤纸味 | 充分 rinse、换 Cafec |
| **风味平、就是"白开水咖啡"** | 水质 TDS 过高或过低 | 配水 / RO + 矿化 |
| **酸度被"压"住，咖啡发闷** | 碱度过高 | 降碱度、换水 |
| **杯子冷了明显变差** | 萃取边缘欠萃 | 均匀注水、提升均粉 |

---

**本章要点**

- Channeling 分侧边和中心两种，修正方式不同
- 读粉床：平整 = 正常；凹坑、划痕、干块各有诊断意义
- Drawdown < 30s 多半研磨粗 / 水温低 / 粉少；> 60s 多半研磨细 / fines 多 / 搅拌过度
- Fines migration 后半段堵塞是最容易忽视的问题，减搅拌 + 过筛
- 刀盘钝化的味觉信号：味道"灰"、苦升甜降、drawdown 变慢
- 滤纸味务必 rinse，浅烘考虑 Cafec Abaca
- 高海拔水温调整、研磨细、时间长
- Muddy 味成因复杂，按 水 → 磨 → 温 → 豆 顺序排查
