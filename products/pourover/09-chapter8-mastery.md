# 第八章 从熟练到精通

到了这一章，你已经有足够的**知识储备**。但"懂"和"精通"之间还隔着一段路——这段路不是靠更多知识走完的，而是靠**系统的练习、记录、反思**。最后这一章讨论的不是技术，而是**如何练习**。

## 如何校准你自己的磨豆机

**前提**：每一台磨豆机都有个体差异，社区刻度表只是参考。**你要建立的是属于你这台机器的刻度 → 微米 → 风味**的对应关系。

### 方法一：TDS 校准

1. 选一支稳定的豆子（建议买 500g 同批次）
2. 固定所有其他参数：粉量、水量、水温、手法
3. 从你认为"适中"的刻度开始，每次变化 2 格（细化 → 粗化）
4. 每杯测 TDS，计算 EY
5. 画出**刻度 → EY 曲线**

理想：你应该得到一条**接近线性**的曲线。某些刻度如果 EY 异常（跳变），说明这个刻度附近 PSD 异常，需要避开。

### 方法二：盲杯三角测试

1. 同一支豆子、同一参数、三个研磨刻度（比如 22、24、26 格）
2. 请朋友（或你自己蒙眼）从三杯里挑"不一样的那杯"
3. 分辨阈值：**通常 2 格以内人脑辨别率 < 70%**
4. 找到"**最小可辨差距**"——这是你磨豆机在你感官上的**实际分辨率**

### 方法三：Drawdown 时间校准

固定其他参数，**调整研磨直到 drawdown 落在目标时间**（如 3:30）。这是最粗但最快的方法，每换一支新豆都可重复一次。

## 感官训练：triangle test 和 cupping

### Triangle Test（三角测试）

最经典的感官辨别训练：

1. 准备**三杯**咖啡：两杯完全相同（A），一杯不同（B）
2. 随机摆放
3. 被测者挑出"不一样"的那杯

**难度分级**：
- 入门：两支不同产区（哥伦比亚 vs 埃塞），90%+ 能选对
- 中级：两种处理法同一产区（水洗 vs 日晒耶加），70–85%
- 高级：**同产区同处理法不同庄园**，50–65%
- 职业：同批豆不同研磨度（1 格差），30–50%

**频率**：每周 1–2 次，半年后明显提升。

### Cupping（杯测）

SCA 杯测协议：
- 粉水比 1:18.18（8.25g 粉 / 150ml 水）
- 研磨：中粗
- 水温 93℃
- 闷蒸 4 分钟
- "破渣"（crust break）→ 撇渣 → 待液体降温至可入口品尝
- 按 SCA 杯测表打分（10 项 × 10 分 = 100 分）

杯测的价值**不在于打分**，而在于**让你在无萃取变量干扰的情况下专注咖啡本身的风味**。每周杯测 1 次 3–4 支豆子，味觉基准会快速建立。

### 风味轮（Flavor Wheel）的使用

SCA 2016 风味轮有约 85 个风味词。不要"想着"找词，而要：

1. 喝一口，**停下来**
2. 问自己："像什么？"（食物、水果、花、香料）
3. 从风味轮中心往外扩展：水果 → 浆果 → 蓝莓 / 草莓 / 黑莓
4. **不确定时就只说"水果"**——模糊不是错，**编造**才是错

## Brew Journal：记录什么

你可以用纸本、Excel、Notion、或专用 App（Beanconqueror）。**核心字段**：

| 字段 | 示例 |
|---|---|
| 日期 | 2026-04-20 |
| 豆子（品名 / 烘焙日期 / 烘焙度） | 埃塞 Yirgacheffe G1 Natural / 2026-04-12 / 浅烘 |
| 粉重 / 水量 / 比例 | 15g / 250g / 1:16.7 |
| 水温 / 水质 | 93℃ / Third Wave Water |
| 磨豆机 / 刻度 | 1Zpresso JX-Pro / 外环 2 + 内环 80 |
| 滤杯 / 滤纸 | V60 02 陶瓷 / Cafec Abaca |
| 手法 | Hoffmann 法 / 2 次 swirl |
| 时间 | 闷蒸 45s / 注水至 1:15 / drawdown 3:45 |
| TDS / EY | 1.28% / 20.1% |
| **风味笔记** | 柑橘（柠檬皮）、茉莉花、蜂蜜、余韵偏短 |
| **评分（1–10）** | 7.5 |
| **下次调整** | 研磨细 1 格、水温 94℃ |

**坚持 3 个月**，你会看到自己的**进步曲线**，并发现自己的**偏好盲点**（比如你其实更爱中烘而不是浅烘，但一直在执着浅烘）。

## 何时升级器具（和何时不必）

### 明确需要升级的信号

- **刀盘钝化**，新刀盘能立即还原（见第七章）
- **磨豆机 PSD 严重双峰**，尤其 fines 超标 → 升级到平刀
- **水壶无温控**或温控偏差 > 2℃
- **秤响应慢**（> 500ms），导致注水超量

### 不建议升级的信号

- **"新机会更好"的 FOMO**：你现有器具还没榨干它的潜力
- **社交媒体炫耀**：滤杯换了 8 个，味觉判断力没进步
- **每次问题归咎于器具**：99% 的问题是**你的技术 + 豆子 + 水**，不是器具

### 边际收益递减曲线

- **0 → ¥500**：基础器具（手磨、鹅颈壶、秤、滤杯），从"不能喝"到"可以喝"
- **¥500 → ¥3000**：中端（JX-Pro、Stagg EKG、陶瓷 V60 等），"可以喝"到"挺好喝"
- **¥3000 → ¥10000**：高端（DF64、Fellow Stagg Pro、Acaia），"挺好喝"到"稳定好喝"
- **¥10000+**：旗舰（Weber Key、EK43），**边际收益极小**，除了比赛/咖啡馆场景收益不大

绝大多数家庭玩家**最佳性价比区间在 ¥2000–5000 之间**。

## 比赛冲煮的真实面貌

World Brewers Cup 不是"冲一杯给评审喝"这么简单：

- **每位选手 10 分钟** 冲**3 杯**（评审各一杯）+ 自述 + 解释
- **评审盲品**，不知道选手用的豆子、参数
- **评分项**：风味、口感、余韵、平衡、整体评价、宾客体验
- **豆子是自选的**——很多冠军豆价格数百美元每磅
- **参数精确到秒和克**，练习数千次后仍会现场紧张导致偏差

**对家庭玩家的启示**：
1. 冠军们的"完美一杯"是**数百次练习的平均水平**，不是一次性天才
2. 比赛是**表演艺术 + 技术**的结合，不能完全复制到家
3. **你不需要做到冠军那一步**，但他们的流程（参数记录、盲测、迭代）你可以学

## 关于精通：不追求完美，追求稳定

新手追问"最好的配方是什么？"——他希望有一个答案。
进阶者追问"这杯为什么这样？"——他在学习诊断。
**精通者**追问"我能不能每次都稳定冲到我想要的 90 分？"——他明白了完美只是偶然。

精通的本质不是**一次 95 分**，而是**每次都至少 85 分**、**波动小于 5 分**。这需要：

- **理解每一个变量的影响**（前七章内容）
- **严格控制你能控制的**（水、粉、温度、手法）
- **接受你不能控制的**（豆子批次差异、天气、心情）
- **做记录、复盘、迭代**

> 做一万次手冲，你不会每次都冲出好咖啡。但你会在第一口时就**知道**这杯是 85 分还是 70 分，**知道**该怎么调下一杯。

这种"知道"，就是精通。
