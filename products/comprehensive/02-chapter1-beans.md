# 第一章 从咖啡樱桃到咖啡豆

## 咖啡豆其实是种子

大多数人第一次知道"咖啡豆是一种水果的种子"时，都会感到惊讶。严格来说，咖啡豆是**咖啡树（Coffea）果实的种子**，这种果实因为外形像樱桃，被咖啡行业称为 **咖啡樱桃（Coffee Cherry）**。

每颗咖啡樱桃里通常有**两颗种子**，对称排列。约 5% 的咖啡樱桃只有一颗圆形种子，行业称为 **圆豆（Peaberry）**，风味通常更集中。

## 两大栽培品种：阿拉比卡与罗布斯塔

全球商业化栽培的咖啡主要有两个物种：

### 阿拉比卡（Coffea arabica）

- **占全球咖啡产量约 60%**
- 起源于埃塞俄比亚高地
- 适合 **海拔 900–2000 米** 的种植环境
- 风味层次丰富，酸度明亮，甜感好
- 咖啡因含量较低（约 1.2–1.5%）
- 对病虫害敏感，种植成本高
- **所有精品咖啡几乎都是阿拉比卡**

### 罗布斯塔（Coffea canephora，商名 Robusta）

- **占全球咖啡产量约 40%**
- 原产于非洲中西部
- 适合 **海拔 200–900 米** 的低地环境
- 风味偏苦、木质、橡胶感，甜感弱
- 咖啡因含量高（约 2.2–2.7%，几乎是阿拉比卡的两倍）
- 抗病能力强、产量高、成本低
- **常用于即溶咖啡、意式拼配豆（用于增加 crema 和浓烈感）**

> **一句话区分**：阿拉比卡追求风味，罗布斯塔追求产量和强度。

### 其他小众品种

- **利比里卡（Liberica）**：东南亚少数产区种植，产量极低
- **艾克塞尔萨（Excelsa）**：2006 年后被并入利比里卡种，偶尔在越南出现

## 阿拉比卡之下的品种

阿拉比卡本身又有数百个亚品种（varieties），精品咖啡圈里最常见的：

| 品种 | 产地代表 | 风味特征 |
|---|---|---|
| **Typica 铁皮卡** | 全球广泛 | 温和平衡，经典风味 |
| **Bourbon 波旁** | 中南美 | 甜感突出，果香 |
| **Geisha / Gesha 瑰夏** | 巴拿马、埃塞 | 茉莉花、佛手柑、极高酸质 |
| **SL28 / SL34** | 肯尼亚 | 黑加仑、番茄、浓郁酸甜 |
| **Caturra 卡杜拉** | 哥伦比亚、巴西 | 果酸明显，产量高 |
| **Pacamara 帕卡玛拉** | 萨尔瓦多 | 大颗粒豆，风味独特 |
| **SL28** | 肯尼亚 | 黑醋栗与黑加仑风味典范 |

**瑰夏（Gesha）** 值得特别说明：2004 年巴拿马翡翠庄园（Hacienda La Esmeralda）用瑰夏拿下 BOP 冠军后，这个品种就成为精品咖啡的代名词。优质瑰夏在拍卖会上可以拍到 **每磅数千美元**，一杯约 20–30ml 的瑰夏 espresso 在精品咖啡馆能卖到 **80–200 元人民币**。

## 从樱桃到生豆：处理法

采摘下来的咖啡樱桃需要"处理"——把果肉、果皮、黏液去除，留下干燥后的种子。处理法对最终风味影响**极大**，甚至超过品种。

### 四种主流处理法

#### 1. 水洗处理法（Washed / Wet Process）

**流程**：
1. 采摘樱桃 →
2. 去除果皮果肉（果肉去除机）→
3. 在发酵槽中浸泡 12–36 小时，微生物分解残留黏液 →
4. 清水冲洗 →
5. 日晒或机械烘干至含水率 10–12%

**风味特征**：
- **干净（Clean）** 是关键词
- 酸度明亮、清晰
- 风味层次分明，没有发酵感
- 典型产区：中美洲（危地马拉、哥伦比亚、哥斯达黎加）、肯尼亚

#### 2. 日晒处理法（Natural / Dry Process）

**流程**：
1. 采摘樱桃 →
2. 直接整颗摊晒在晒床或水泥地上 14–30 天 →
3. 期间不断翻动防止发霉 →
4. 干燥至合适含水率后脱壳

**风味特征**：
- **甜感强烈**（果肉的糖分被种子吸收）
- 浆果、红酒、发酵感
- 风味复杂但有时显"野"
- 典型产区：埃塞俄比亚（耶加雪菲日晒）、巴西

#### 3. 蜜处理法（Honey Process）

**流程**：
1. 去除果皮 →
2. 保留不同比例的黏液（Mucilage）→
3. 晾晒 10–20 天

根据保留黏液的多少，分为：

| 类型 | 黏液保留 | 风味 |
|---|---|---|
| **白蜜 White Honey** | 很少 (~10%) | 接近水洗 |
| **黄蜜 Yellow Honey** | 约 25% | 干净但带甜 |
| **红蜜 Red Honey** | 约 50% | 甜感明显 |
| **黑蜜 Black Honey** | 80–100% | 接近日晒 |

典型产区：哥斯达黎加、巴拿马。

#### 4. 厌氧发酵处理法（Anaerobic Fermentation）

**流程**：
1. 樱桃装入密封发酵罐 →
2. 在缺氧环境下发酵 48–120 小时 →
3. 风味由发酵过程决定

**风味特征**：
- **强烈的发酵风味**：威士忌、肉桂、菠萝、热带水果
- 风格鲜明，爱恨分明
- 产量小、价格高
- 典型产区：哥伦比亚、哥斯达黎加、云南部分精品庄园

> **新手建议**：先从**水洗处理**开始，建立对"干净咖啡"的味觉基准。然后再去尝试日晒、蜜处理、厌氧，分辨它们带来的差异。

## 咖啡的新鲜度与储存

**烘焙后的咖啡豆有保质期，但与包装上的"保质期 24 个月"关系不大。**

真正决定风味的是**风味巅峰期**：

| 时间段 | 状态 |
|---|---|
| 烘焙后 0–3 天 | 排气期，风味尚未稳定 |
| **烘焙后 4–21 天** | **风味巅峰** |
| 烘焙后 21–60 天 | 风味开始衰减但仍可接受 |
| 烘焙后 60 天以上 | 明显陈化，酸质下降，木质感上升 |

**储存要点**：
- 使用**不透明密封罐**（有单向排气阀更佳）
- 避光、干燥、常温（15–25℃）
- **不要放冰箱**：反复进出温差导致水汽凝结
- **不要放冷冻**：除非买了大批量要分装冻存，解冻后只用一次

**研磨后的咖啡**氧化速度比整豆快 **30 倍以上**，所以始终坚持 **现冲现磨**。

---

**本章要点**

- 咖啡豆是咖啡樱桃的种子，最重要的两大品种是阿拉比卡（品质高）和罗布斯塔（成本低）
- 处理法（水洗/日晒/蜜处理/厌氧）对风味影响巨大，新手从水洗入门
- 精品咖啡的风味巅峰在烘焙后 4–21 天，过了两个月基本失味
