# 第六章 手冲咖啡：V60、Kalita、Origami 三大滤杯

手冲（Pour Over / Filter Coffee）是**精品咖啡最常用的冲煮方式**。它便宜（200 元能配齐全套）、稳定（参数可复现）、风味干净（滤纸过滤油脂）、信息量大（一杯手冲能喝出豆子的产地处理法）。

本章教你从零开始做出一杯水准稳定的手冲，并掌握三大主流滤杯的差异。

## 手冲的基本原理

手冲是一种**浸渍 + 过滤并存**的萃取方式：

1. 热水与咖啡粉接触，溶出可溶物质
2. 咖啡液在重力作用下通过粉床
3. 滤纸拦截咖啡油脂与细粉，得到干净的咖啡液

所有手冲器具的差异，本质上只是**粉床形状 + 出水速度 + 水粉接触时间** 的不同组合。

## 三大主流滤杯

### Hario V60（锥形 / 单孔）

日本 Hario 公司 2004 年推出，是目前全球普及率最高的滤杯。

- **形状**：60° 圆锥，单一大孔
- **内壁**：螺旋导流肋（Spiral Rib），让空气逸出、避免滤纸贴壁
- **流速**：**快**，高度依赖手法
- **风味风格**：干净、层次丰富、突出酸质与风味层
- **适合**：浅到中烘单品豆、技术派爱好者
- **代表规格**：01（1–2 杯，适合 10–15g 粉）、02（1–4 杯，15–30g 粉）

V60 是一只"手法放大器"——冲得好非常好，冲得差也非常明显。

### Kalita Wave（平底 / 三孔）

日本 Kalita 公司推出的平底波纹滤杯。

- **形状**：平底，三个均匀分布的小孔
- **滤纸**：特制波浪纸 Wave Filter，固定的 20 个褶皱
- **流速**：**中等偏稳**，水粉接触更均匀
- **风味风格**：平衡、稳定、甜感突出
- **适合**：中烘豆、奶咖做单品、追求稳定性的家庭用户
- **代表规格**：155（1–2 杯）、185（2–4 杯）

Kalita Wave 是"容错率高"的代名词。手抖、水流不稳，它都能给你一杯及格的咖啡。

### Origami 折纸滤杯

日本岐阜美浓烧陶瓷品牌 Trinity 推出的"万能型"滤杯，被 Kasuya Tetsu 等知名咖啡师推广使用。

- **形状**：20 面折纸造型，内壁立体支撑
- **滤纸兼容**：**同时兼容 V60 锥形滤纸与 Kalita Wave 波纹纸**
- **流速**：配锥形纸偏快（接近 V60），配波纹纸偏稳（接近 Kalita）
- **风味风格**：根据配纸切换，灵活
- **适合**：不想买多个滤杯、希望一杯多用的爱好者
- **代表规格**：S（1–2 杯）、M（1–4 杯）

### 三者对比

| 维度 | V60 | Kalita Wave | Origami |
|---|---|---|---|
| 底部 | 单大孔 | 三小孔 | 单大孔 |
| 形状 | 60° 锥形 | 平底 | 20 面折纸 |
| 流速 | 快 | 中等 | 可调 |
| 容错 | 低 | **高** | 中 |
| 风味风格 | 层次、锐利 | 平衡、甜 | 灵活 |
| 适合人群 | 进阶、技术派 | **新手首选** | 折衷派 |

## 手冲器具清单

一套最小可用组合（预算 500 元内）：

- 滤杯：V60 01（约 80 元，塑料版性价比最高）
- 滤纸：原厂 V60 滤纸（漂白版 / 本色版均可）
- 分享壶 / 下壶：玻璃 400ml
- **手冲壶**：细口鹅颈壶，400–600ml。Hario Buono、Kalita Pelican、Fellow Stagg EKG（带温控）
- **磨豆机**：**这是最关键的投入**。1Zpresso Q2 / Timemore C3 起步
- **电子秤**：0.1g 精度，带计时功能。HARIO V60 Drip Scale、Timemore Black Mirror
- **温度计**（可选）：烧水壶有温控更方便

## 4:6 法（Kasuya Tetsu 大桥技术）

2016 年粕谷哲（Kasuya Tetsu）用这个方法拿下世界手冲大赛冠军，此后成为最广泛传播的 V60 配方之一。

**核心思想**：把总水量分成两部分——**前 40% 控制酸甜平衡**，**后 60% 控制浓度**。

### 4:6 标准配方

- 粉量：20g
- 水量：300g（1:15）
- 水温：93℃
- 研磨度：中等偏粗

分 5 段注水，每段 60g：

| 段数 | 水量 | 时间点 | 作用 |
|---|---|---|---|
| 1 | 0 → 60g | 0:00 → 0:45 | **闷蒸 + 第 1 段** 控酸甜（少 = 更甜，多 = 更酸） |
| 2 | 60 → 120g | 0:45 → 1:30 | 第 2 段（前 40% 结束）|
| 3 | 120 → 180g | 1:30 → 2:00 | 后 60% 第 1 段 |
| 4 | 180 → 240g | 2:00 → 2:30 | 后 60% 第 2 段 |
| 5 | 240 → 300g | 2:30 → 3:00 | 后 60% 第 3 段 |

**总时长约 3:30 分钟**（包含最后滴完的时间）。

调整方向：

- 想更甜 → 第 1 段注 50g、第 2 段注 70g
- 想更酸 → 第 1 段注 70g、第 2 段注 50g
- 想更浓 → 后 60% 分 2 段而不是 3 段
- 想更淡 → 后 60% 分 4 段

## 标准 V60 新手配方

如果你第一次手冲，先别学 4:6。用这个简化版：

### 配方（1 杯量）

- **粉量**：15g
- **水量**：225g（1:15）
- **水温**：**92℃**（中烘）/ 94℃（浅烘）/ 90℃（深烘）
- **研磨度**：中等（粗砂糖质感）
- **滤杯**：V60 01
- **目标总时间**：**2:30**

### 步骤

1. **准备（0:00 前）**
   - 烧水至 95℃（浅烘）
   - 滤纸放入滤杯，用热水充分润湿，**弃掉润纸水**（去除纸味 + 预热下壶）
   - 15g 咖啡豆中研磨，倒入滤杯，轻拍摊平
   - 分享壶归零

2. **闷蒸 Bloom（0:00 → 0:30）**
   - 注水 **30g**（咖啡粉的 **2 倍重**）
   - 以画圆方式由中心向外均匀淋湿所有粉
   - 静置 30 秒（新鲜豆会"膨胀"冒泡，称为"汉堡"）

3. **主萃取第一段（0:30 → 1:10）**
   - 继续注水至 **总量 130g**
   - 中心稳定画圆，由内向外再回到中心
   - 水流细、稳定

4. **主萃取第二段（1:10 → 1:40）**
   - 注水至 **总量 225g**
   - 同样手法

5. **滴滤完成（1:40 → 2:30）**
   - 停止注水，等咖啡液滴完
   - 粉床应呈"平坦的小山丘"——侧面有粉附着、中间干净

如果**超过 3:00 还没滴完** → 下次研磨粗一点
如果**不到 2:00 就滴完** → 下次研磨细一点

## 注水手法：两大流派

### 中心注水（Center Pour）

- 水流只在中心小范围内
- 粉床被"从中心向外推"
- 萃取更强、风味更集中
- 适合浅烘、追求明亮风味

### 螺旋画圆（Spiral Pour）

- 从中心向外螺旋扩散，再收回中心
- 粉床萃取更均匀
- 风味更平衡
- **新手推荐**

**不要冲到滤纸上！** 水流一旦打到滤纸侧壁，会沿纸壁流下、绕过粉床——这叫"边壁效应"，是萃取不足的最常见原因。

## 闷蒸（Bloom）详解

- **定义**：注入咖啡粉重量 **2 倍**的水，静置 **30–45 秒**
- **作用**：
  - 释放豆子内部的 CO₂（新鲜豆子排气量大）
  - 让粉床充分浸润，避免后续注水时出现干粉团
  - 预热粉床
- **现象**：新鲜豆会膨胀成"汉堡"；陈豆基本不膨胀——**闷蒸的膨胀度是豆子新鲜度的可视指标**
- **时间**：新鲜豆 45 秒，一般豆 30 秒，陈豆 15 秒即可

## 常见问题排查

### 通道效应（Channeling）

- **现象**：粉床某一点出现坑/裂缝，水全往那里走
- **原因**：粉床不平、注水冲击过猛、研磨不均
- **解决**：摊平粉床、降低注水高度、用 WDT 针搅拌

### 萃取不均

- **现象**：粉床结束后一边高一边低、侧面很多干粉
- **原因**：注水不对称、水流过粗
- **解决**：保持画圆对称，水流细而稳

### 苦涩

- **原因**：研磨过细 / 水温过高 / 时间过长 / 粉量过多
- 按第四章方法一次调一个

### 尖酸、水感

- **原因**：研磨过粗 / 水温过低 / 时间过短 / 豆子萃取不足
- 细研磨半格、水温提高 2℃ 再试

### 粉床"下陷成坑"

- **原因**：细粉过多、滤杯孔被堵
- **解决**：磨豆机用筛网过筛细粉（可选）、检查磨豆机刀盘

**本章要点**

- V60 锐利层次、Kalita Wave 平衡容错高、Origami 兼容两种滤纸；新手首选 Kalita 或 Origami
- 标准 V60 新手配方：15g / 225g / 92℃ / 1:15 / 总时间 2:30
- 闷蒸用粉重 2 倍的水、静置 30–45 秒；新鲜豆膨胀成"汉堡"
- 水流画圆避开滤纸壁，一次只调一个变量；超过 3 分钟磨粗，不到 2 分钟磨细
