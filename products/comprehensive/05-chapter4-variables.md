# 第四章 萃取的四大核心变量

冲咖啡的本质是 **"用热水把咖啡豆里可溶物质溶解出来"** 的过程。能否稳定做出一杯好咖啡，取决于你对四个核心变量的理解与控制：

1. **研磨度**（Grind Size）
2. **水温**（Water Temperature）
3. **粉水比**（Brew Ratio）
4. **萃取时间**（Extraction Time）

外加两个"结果变量"——**TDS**（总溶解固体）与 **EY**（萃取率）——用来量化你冲出来的是什么东西。

本章是全书最重要的一章。后面所有冲煮方法、所有故障排查，全都回到这一章。

## 变量一：研磨度

研磨度直接决定了 **水与咖啡粉的接触面积**。粉越细，接触面积越大，萃取越快、越充分；粉越粗，接触面积越小，萃取越慢。

### 研磨度与器具的对照

| 研磨度 | 颗粒直径（参考） | 视觉参考 | 适用器具 |
|---|---|---|---|
| 极细 | **< 200 μm** | 面粉 | 土耳其咖啡 |
| 细 | **200–400 μm** | 细砂糖、食盐 | 意式浓缩 Espresso |
| 中细 | **400–600 μm** | 粗砂糖 | 摩卡壶、爱乐压短时间 |
| 中 | **600–800 μm** | 粗盐 | V60、Kalita、Origami 手冲 |
| 中粗 | **800–1000 μm** | 海盐、粗砂糖颗粒 | Chemex、聪明杯 |
| 粗 | **1000–1200 μm** | 小石头、Kosher 盐 | 法压壶 French Press |
| 极粗 | **> 1200 μm** | 大粒海盐 | 冷萃 Cold Brew |

以上只是参考，**不同品牌磨豆机刻度不通用**。在家请以"稳定在某个刻度，然后根据风味微调"作为方法论，不要迷信数字。

### 研磨均匀度（Uniformity）

研磨度的平均值之外，**均匀度** 同样重要。

- 平刀（Flat Burr）：均匀度高，萃取干净，适合手冲与意式
- 锥刀（Conical Burr）：中间偏粗、两端偏细，口感更厚重，适合意式
- 桨式/砍刀式（Blade Grinder）：**极度不均匀，不推荐用于冲煮**

同一份粉里如果细粉（fines）过多，会造成**过萃的苦涩**；粗粉过多则**萃取不足**。均匀度差的磨豆机，哪怕调到理想刻度，杯里也会同时出现过萃与欠萃。

## 变量二：水温

水温决定了**溶解速度**。温度越高，风味物质析出越快，但超过临界点后，苦味与杂味也会加速析出。

### 不同烘焙度的推荐水温

| 烘焙度 | 推荐水温 | 逻辑 |
|---|---|---|
| **浅烘 / 极浅** | **93–96℃** | 豆子密度大、可溶物难溶出，需要高温 |
| **中烘** | **90–93℃** | 折中区间，平衡酸甜 |
| **中深烘** | **88–91℃** | 防止苦味过度析出 |
| **深烘 / 法式** | **85–90℃** | 豆质疏松，低温即可充分萃取，高温反而过苦 |

SCA（精品咖啡协会）官方推荐的冲煮水温是 **92.2–94.4℃**（198–202℉），覆盖绝大多数中到浅烘精品豆。

意式浓缩是例外——水温 **90.5–94℃**，但因为压力与时间极短，实际萃取动力学完全不同。

### 水温的实操提醒

- 沸水（100℃）下壶后，倒入温热的手冲壶，温度会降到约 **95℃**
- 再倒入室温滤杯与分享壶，降到约 **92℃**
- 所以：**冲浅烘用刚沸腾后静置 10–20 秒的水**；**冲深烘把水烧开后敞口等 1–2 分钟**

## 变量三：粉水比（Brew Ratio）

粉水比决定了**浓度**。写法统一为 `1:X`——即 1 份咖啡粉对 X 份水。

### 经典粉水比区间

| 方法 | 推荐比例 | 风格 |
|---|---|---|
| 意式浓缩（Ristretto）| 1:1 – 1:1.5 | 浓烈、甜感集中 |
| 意式浓缩（Normale）| 1:2 | 经典平衡 |
| 意式浓缩（Lungo）| 1:3 – 1:4 | 稀释、风味更开 |
| 摩卡壶 | 1:7 – 1:10 | 浓烈但非 espresso 强度 |
| 爱乐压 | 1:10 – 1:16 | 灵活 |
| **手冲** | **1:15 – 1:17** | **SCA 金杯标准（黄金区间）** |
| 法压壶 | 1:15 – 1:17 | 与手冲相近 |
| 冷萃（浓缩式）| 1:8 – 1:10 | 需加水稀释饮用 |
| 冷萃（即饮式）| 1:12 – 1:15 | 直接饮用 |

### SCA 金杯标准（Golden Cup）

SCA 官方推荐的家庭冲煮基准是：

- **粉水比：55 g/L ± 10%**（约 1:18 ± 10%）
- **TDS：1.15–1.35%**
- **萃取率 EY：18–22%**

欧美 SCA 偏爱 **1:18**，日本/亚洲偏爱 **1:15–1:16**（浓度更高）。选哪个是口味偏好，不是对错。

## 变量四：萃取时间

时间是前三个变量的**结果**，也是最后一个可调项。

### 典型时间区间

| 方法 | 总时间 |
|---|---|
| 意式浓缩 | **25–30 秒** |
| V60 手冲 | **2:00 – 3:30 分钟**（含闷蒸） |
| Kalita 手冲 | 3:00 – 4:00 分钟 |
| Chemex | 4:00 – 5:30 分钟 |
| 爱乐压（标准）| 1:30 – 2:30 分钟 |
| 法压壶 | **4 分钟**（经典） |
| 摩卡壶 | 4–6 分钟（到出液结束）|
| 冷萃 | **12–18 小时 @ 4℃** |

**时间太短** → 萃取不足，酸涩、尖锐、水感
**时间太长** → 过度萃取，苦涩、干涩、涩口

## 结果变量：TDS 与 EY

这两个是喝到嘴里的"客观答案"。

### TDS（Total Dissolved Solids，总溶解固体）

咖啡液中**溶解物质的重量百分比**。用数字咖啡折光仪（Refractometer，如 VST、Atago Coffee）可直接读出。

| 饮品 | 典型 TDS |
|---|---|
| 手冲咖啡 | **1.15 – 1.55%** |
| 法压 | 1.20 – 1.60% |
| 意式浓缩 | **8 – 12%** |
| 冷萃（浓缩）| 2.5 – 4.0% |

TDS 反映**浓度**——喝起来"稀"还是"厚"。

### EY（Extraction Yield，萃取率）

咖啡粉中**被溶解出来的物质占原始粉重的百分比**。计算公式：

```
EY (%) = (咖啡液重量 × TDS) / 咖啡粉重量
```

| EY 区间 | 风味表现 |
|---|---|
| **< 18%** | 萃取不足：酸涩、咸、尖锐、水感 |
| **18 – 22%** | **理想区间（SCA Golden Cup）** |
| **> 22%** | 过度萃取：苦、干、涩、木质 |

新手不必买折光仪——**先靠味觉训练**。你能分清"萃取不足的尖酸"和"萃取合理的果酸"时，折光仪才有意义。

## 萃取平衡模型：酸–甜–苦

咖啡粉内部物质的**溶出顺序**大致是：

1. **酸类**（最先溶出）——柠檬酸、苹果酸、奎宁酸
2. **糖与甜感物质**
3. **苦味物质**（最后溶出）——绿原酸降解产物、咖啡因、焦油

这解释了一切：

- 萃取不足 = 只拿到了酸，没有甜和苦 → 尖酸
- 萃取适中 = 酸、甜、苦比例协调 → 平衡
- 过度萃取 = 酸已经挥发或被平衡，苦味占主导 → 苦涩

**好咖啡的目标：拿到足够多的甜，一点点苦平衡整体，酸作为亮色。**

## 变量之间的互相影响（最重要的一张表）

一次只动一个变量，其他保持不变：

| 现象 | 调整方向 |
|---|---|
| 太酸、尖锐 | **研磨更细** / 水温更高 / 时间更长 |
| 太苦、涩口 | 研磨更粗 / 水温更低 / 时间更短 |
| 太淡、水感 | 提高粉量（改变比例，如 1:18 → 1:15） |
| 太浓、粘稠 | 降低粉量（1:15 → 1:17） |
| 风味空洞、没亮点 | 换豆 / 换更新鲜的豆 / 换处理法 |

---

**本章要点**

- 四大变量：研磨度、水温、粉水比、时间；结果变量：TDS 与 EY
- 浅烘用 93–96℃，中烘 90–93℃，深烘 85–90℃；手冲粉水比 1:15–1:17（SCA 1:18）
- 理想萃取率 EY 18–22%，TDS 1.15–1.55%（手冲）
- 太酸 → 细磨 / 高温 / 长时；太苦 → 粗磨 / 低温 / 短时；一次只改一个变量
